Tags Summary

This page demonstrates all inline text formatting tags:

  • <a> - Anchor/link elements
  • <strong> - Strong importance (semantic)
  • <em> - Emphasized text (semantic)
  • <b> - Bold text (presentational)
  • <i> - Italic text (presentational)
  • <u> - Underlined text
  • <small> - Small text
  • <mark> - Highlighted/marked text
  • <sub> - Subscript text
  • <sup> - Superscript text
  • <abbr> - Abbreviations
  • <cite> - Citations
  • <code> - Code snippets
  • <span> - Generic inline container
  • <time> - Time/date elements
  • <var> - Variables
  • <kbd> - Keyboard input
