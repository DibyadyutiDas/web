Tags Summary

This page demonstrates all text content tags:

  • <p> - Paragraphs
  • <br> - Line breaks
  • <hr> - Horizontal rules
  • <blockquote> - Block quotations
  • <q> - Inline quotations
  • <cite> - Citations
  • <pre> - Preformatted text
  • <ol> - Ordered lists
  • <ul> - Unordered lists
  • <li> - List items
  • <dl> - Description lists
  • <dt> - Description terms
  • <dd> - Description definitions
