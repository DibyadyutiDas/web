Meta Tags Used in This Document

  • charset="UTF-8" - Character encoding
  • viewport - Responsive design settings
  • description - Page description for search engines
  • keywords - Keywords for SEO (less important now)
  • author - Page author information
  • robots - Search engine indexing instructions
  • og:title, og:description, og:type - Open Graph for social media
